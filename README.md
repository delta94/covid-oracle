# covid-oracle

### How to use 

<ol>
<li>Clone or download source code</li>
<li>Run 'npm install' to install all package</li>
<li>Change your config database in ./configs/dbconfig/</li>
<li>Run 'node index' to start project</li>
</ol>

### Enviroment

<ul>
<li>Node.js</li>
<li>Oracle database</li>
</ul>
